# Support Race Assistant

Race Assistant exists to give community races, schools, clubs, and nonprofits
a capable timing system without a subscription. Sponsorship helps cover code
signing, test equipment, hosting, and the time required to test race-day
workflows and maintain releases.

## Suggested GitHub Sponsors tiers

These are starting points for the GitHub Sponsors profile:

- **$5/month — Community Supporter:** Help keep Race Assistant free and
  maintained, with supporter updates through GitHub Sponsors.
- **$25/month — Race Director Supporter:** For race directors and clubs that use
  Race Assistant and want to sustain continued testing and documentation.
- **$100/month — Project Sponsor:** For organizations that want to support free,
  open-source timing. With permission, the organization may be thanked by name
  in project sponsor acknowledgements.
- **One-time sponsorship:** A simple way to contribute after a successful race
  or fund a specific piece of test hardware.

Sponsorship supports the project; it does not purchase roadmap control,
guaranteed support response times, endorsement, or trademark rights. Paid event
timing, setup help, training, and custom development can be arranged separately
when available.

## Other ways to help

- Test a full race workflow before release day and report reproducible issues.
- Improve documentation or share a sanitized import template.
- Tell other community race directors about the project.
- Contribute a focused fix or accessibility improvement.

Thank you for helping make dependable race timing more accessible.
